##  Overview

This portfolio website serves as a central hub for showcasing my technical skills, projects, and professional experience. Built with clean, semantic HTML, modern CSS, and vanilla JavaScript, it demonstrates my understanding of web development fundamentals and responsive design principles.

**Live Site:** https://Shobinn24.github.io/Portfolio_Project_Lab_2

##  Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Smooth Navigation** - Sticky navigation bar with smooth scrolling to sections
- **Modern UI/UX** - Clean, professional design with gradient accents
- **Interactive Elements** - Hover effects, scroll animations, and active link highlighting
- **Project Showcase** - Detailed project cards with links to GitHub repositories
- **Skills Grid** - Organized display of technical competencies
- **Contact Information** - Easy access to email, LinkedIn, and GitHub profiles
- **Performance Optimized** - Lightweight vanilla JavaScript, no frameworks needed

##  Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom properties, flexbox, grid layout, animations
- **Vanilla JavaScript** - Smooth scrolling, intersection observers, DOM manipulation
- **Git & GitHub** - Version control and deployment

##  Installation & Usage

### View Locally

1. Clone the repository:
```bash
git clone https://github.com/Shobinn24/Portfolio_Project_Lab_2.git
```

2. Navigate to the project directory:
```bash
cd Portfolio_Project_Lab_2
```

3. Open `index.html` in your browser:
```bash
# On Mac
open index.html

# On Windows
start index.html

# Or simply double-click index.html
```

### Deploy to GitHub Pages

1. Go to your repository settings on GitHub
2. Navigate to "Pages" section
3. Select your main/master branch as the source
4. Your site will be live at `https://shobinn24.github.io/Portfolio_Project_Lab_2`

##  Design Highlights

### Color Palette
- **Primary Blue:** #2563eb - Used for accents and links
- **Secondary Green:** #10b981 - Used for highlights and success states
- **Gradient Hero:** Purple to pink gradient (#667eea to #764ba2)
- **Neutral Grays:** Multiple shades for text and backgrounds

### Typography
- **Font Family:** Segoe UI (system font stack for performance)
- **Headings:** Bold, hierarchical sizing
- **Body Text:** 1.1rem with 1.6 line-height for readability

### Layout
- **CSS Grid:** For skills and projects sections
- **Flexbox:** For navigation and buttons
- **Container Max-Width:** 1200px for optimal reading experience
- **Responsive Breakpoints:** 768px and 480px
