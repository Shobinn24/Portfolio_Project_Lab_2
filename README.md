# Personal Portfolio Website

A responsive portfolio website showcasing my projects, skills, and contact information.

## Project Overview 

This is a simple, clean, and professional portfolio website built with HTML and CSS.

## Features

Adapts to all screen sizes (desktop, tablet, mobile)
Direct links to email, Github and Linkedin
Hover effects

## Technologies Used

HTML
CSS

# HTML Structure
<Headers> - Name and Title
<Main> - Main content
<Section> - About Me, Projects, and Contact Sections
<Footer> - Copyright information
